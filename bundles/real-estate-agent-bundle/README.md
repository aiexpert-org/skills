## The thirteen skills

| # | Skill | What it does |
|---|---|---|
| 1 | `showing-prep` | Prep the second-and-subsequent showing. |
| 2 | `mls-follow-up` | Sequence the property-preference match sends between tours. |
| 3 | `deal-doc-review` | Review contracts, addenda, and closing docs for red flags. |
| 4 | `referral-tracking` | Track and nurture the agent's referral network. |
| 5 | `transaction-milestones` | Milestone-by-milestone checklist from offer to closing. |
| 6 | `listing-presentation` | Draft the listing presentation. |
| 7 | `buyer-consultation` | Structure the initial buyer consultation. |
| 8 | `open-house-plan` | Pre-event plan for the open house. |
| 9 | `neighborhood-brief` | Recent sales, schools, comparable inventory. |
| 10 | `fsbo-outreach` | Outreach sequence for FSBO listings. |
| 11 | `agent-weekly-review` | Agent-tier weekly review. |
| 12 | `open-house-lead-capture` | **NEW v1.1.0.** Digital-first sign-in, same-day follow-up, 7-day nurture, 4-way triage. |
| 13 | `first-showing-checklist` | **NEW v1.1.0.** Pre-showing prep, observation prompts, post-showing debrief, offer-readiness categorization. |
