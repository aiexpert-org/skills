# AI Expert Real Estate Operator Bundle

Thirteen Claude skills for residential real estate agents. Showing prep, listing presentation, transaction milestones, FSBO outreach, buyer consultation, open house plan, neighborhood brief, deal-doc review, and referral tracking.

## Install

Install from Claude Code:
  /plugin marketplace add aiexpert-org/skills
  /plugin install real-estate-operator@aiexpert-org

Install from Claude.ai or Claude Desktop:
  Download the bundle zip from https://github.com/aiexpert-org/skills/releases
  Settings then Skills then Add then select the zip.

## Why this is free

This bundle is free because it is not the product. The product is the working relationship. If these skills change how you run your real estate business, the next question is usually "what else can we do with this?" That is the conversation the AI Expert Fractional CAIO retainer was built for.

Free skills. Newsletter. Then a real conversation when you are ready.

## The 13 skills

1. `showing-prep`. Prep a specific property showing: comps, buyer profile, objections, closing angles.
2. `mls-follow-up`. Post-showing and post-open-house follow-up sequences with drafted messages and cadence.
3. `deal-doc-review`. Review a purchase agreement, addendum, or listing agreement for standard vs unusual terms.
4. `referral-tracking`. Maintain the referral pipeline with sent, received, thank-you cadence, partner health.
5. `transaction-milestones`. Run a transaction from mutual acceptance to close with per-milestone client updates.
6. `listing-presentation`. Draft a listing presentation with CMA, marketing plan, timeline, expected outcome range.
7. `buyer-consultation`. Buyer intake and canonical buyer profile every subsequent skill reads.
8. `open-house-plan`. Pre-event promotion, day-of setup and hosting script, post-event follow-up.
9. `neighborhood-brief`. Neighborhood brief: schools, comps, market temp, HOA, commute, community vibe.
10. `agent-weekly-review`. Weekly roll-up: active deals, pipeline movement, referrals, next week's calendar.
11. `fsbo-outreach`. FSBO discovery, outreach, and nurture cadence with per-FSBO tracking.
12. `handoff`. Hand off a slice to a TC, ISA, broker, or vendor with a scoped briefing doc.
13. `hoard`. Capture information into your vault, tagged for retrieval.
