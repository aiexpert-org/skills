## What this is

The free universal foundation. Seventeen skills that make Claude work the way Anthropic recommends and the way AI Expert extends. Session handoffs, snippet hoarding, dream escalation, daily and weekly reviews, decision logging, context management, project scaffolding, voice-corpus building, voice-input protocol, domain glossary building, CLAUDE.md and LIVING-NOTES.md hygiene, plus the four new v1.1.0 additions: human approval gate, context rot check, install Anthropic defaults, and Claude account audit.

## Anthropic Recommends

The Essential Bundle installs Anthropic's default posture:

- `human-approval-gate` for irreversible actions.
- `install-anthropic-defaults` walks the operator through tool permissions, MCP set, session hooks.
- `claude-account-audit` verifies the install matches Anthropic and AI Expert baselines.
- `context-rot-check` detects when a session should be handed off.
