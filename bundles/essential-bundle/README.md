## Prerequisite

## What this is

The free universal foundation every operator installs on their Claude before choosing a persona layer. Seventeen skills that make Claude work the way Anthropic recommends and the way AI Expert extends.

Session handoffs, snippet hoarding, dream escalation, daily and weekly reviews, decision logging, context management, project scaffolding, voice-corpus building, voice-input protocol, domain glossary building, CLAUDE.md and LIVING-NOTES.md hygiene, plus the four new v1.1.0 additions: human approval gate, context rot check, install Anthropic defaults, and Claude account audit.

## Anthropic Recommends

The Essential Bundle installs Anthropic's default posture for agentic Claude out of the box:

- **`human-approval-gate`** — the Anthropic-recommended human-in-the-loop checkpoint for any action with real-world consequences (payments, sends, deletes, deploys, contract signatures).
- **`install-anthropic-defaults`** — walks the operator through the tool permission allowlist, the recommended MCP set for solo operators, the session hooks Anthropic recommends, and the Anthropic-authored skills to install first.
- **`claude-account-audit`** — verifies the operator's install matches both Anthropic's baseline and the AI Expert baseline.
- **`context-rot-check`** — detects when a long session has decayed to the point that a fresh session would be higher leverage.

Anthropic ships defaults most operators never turn on. The Essential Bundle turns them on.

## Install

Three paths.

**Claude Code plugin marketplace.**

```
/plugin marketplace add aiexpert-org/skills
/plugin install essential-bundle@aiexpert
```

**Claude Desktop app.**

1. Download `essential-bundle-v1.1.0.zip` from the newsletter link.
2. Unzip.
3. Claude Desktop, Settings, Skills, Add Skill, Upload folder.
4. Point at the `skills/` folder inside the unzipped bundle.
5. Enable all 17 skills.

**Curl script.**

```
curl -sL https://install.aiexpert.org/essential | bash
```
