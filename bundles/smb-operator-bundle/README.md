# AI Expert SMB Operator Bundle

Twelve Claude skills for small business operators. Sales pipeline hygiene, cold outreach discipline, meeting prep, weekly review, decision log, brand voice enforcement, GHL workflow templates, and end-of-day close-out.

## Install

Install from Claude Code:
  /plugin marketplace add aiexpert-org/skills
  /plugin install smb-operator@aiexpert-org

Install from Claude.ai or Claude Desktop:
  Download the bundle zip from https://github.com/aiexpert-org/skills/releases
  Settings then Skills then Add then select the zip.

## Why this is free

This bundle is free because it is not the product. The product is the working relationship. If these skills change how you run the business, the next question is usually "what else can we do with this?" That is the conversation the AI Expert Fractional CAIO retainer was built for.

Free skills. Newsletter. Then a real conversation when you are ready.

## The 12 skills

1. `weekly-network-review`. Sunday reach-out list from the operator's CRM.
2. `sales-pipeline-hygiene`. Weekly pipeline scan for stuck, nurture, kill, forecast.
3. `cold-outreach-discipline`. Sender reputation guardrails before any cold campaign.
4. `brand-voice-consistency`. Voice check on any draft, plus rewrite in voice.
5. `ghl-workflow-templates`. Ready-made GoHighLevel workflows for common SMB flows.
6. `meeting-prep`. One-page brief for any meeting worth preparing for.
7. `end-of-day-review`. Daily close-out with tomorrow's top three.
8. `decision-log`. Captures the decision, alternatives, rationale, review date.
9. `hoard`. Save a solved-problem snippet for future search.
10. `handoff`. Handoff doc for a teammate, agent, or future self.
11. `weekly-review`. Friday umbrella review across pipeline, network, decisions.
12. `ghost-writer-me`. First-pass draft of anything in the operator's voice.
