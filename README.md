# AI Expert Marketplace

Claude Mastery skill bundles from AI Expert. Free Essential Bundle plus five persona layers.

## Bundles

- **[Essential Bundle](bundles/essential-bundle/)** — 17 universal skills. The free foundation every operator installs. Includes Anthropic-default posture skills (`human-approval-gate`, `install-anthropic-defaults`, `context-rot-check`, `claude-account-audit`).
- **[Small Business Owner Bundle](bundles/small-business-owner-bundle/)** — 12 skills for solopreneurs through 25-employee businesses.
- **[Executive Bundle](bundles/executive-bundle/)** — 16 skills for VP through C-suite.
- **[Real Estate Agent Bundle](bundles/real-estate-agent-bundle/)** — 13 skills for solo agents through 5-agent teams.
- **[Author Bundle](bundles/author-bundle/)** — 12 skills for nonfiction authors before, during, and after launch.
- **[Podcast Host Bundle](bundles/podcast-host-bundle/)** — 12 skills for authenticity-focused hosts.

## Install (Claude Code)

```
/plugin marketplace add aiexpert-org/skills
/plugin install essential-bundle@aiexpert
/plugin install <persona-bundle>@aiexpert
```
