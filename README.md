# AI Expert Marketplace

The first Claude skill library built for operators, not engineers.

Free bundles of Claude skills for the people running the business. Small business owners, mid-market executives, real estate agents. Every bundle installs in one command, fires on how you actually talk, and produces output you can use without editing. Built by Brett K. Moore.

## Bundles

### SMB Operator Bundle
12 skills for small business operators: sales pipeline hygiene, cold outreach discipline, meeting prep, weekly review, decision log.
```
/plugin marketplace add aiexpert-org/skills
/plugin install smb-operator@aiexpert-org
```

### Executive Bundle
14 skills for mid-market executives: the 5-Stage Operator Lens, dream escalation, board deck draft, investor update, hiring loop.
```
/plugin marketplace add aiexpert-org/skills
/plugin install executive@aiexpert-org
```

### Real Estate Operator Bundle
13 skills for residential real estate agents: showing prep, listing presentation, transaction milestones, FSBO outreach.
```
/plugin marketplace add aiexpert-org/skills
/plugin install real-estate-operator@aiexpert-org
```

For Claude.ai and Claude Desktop install paths, see `docs/install.md`.
